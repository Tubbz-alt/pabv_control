# Installation
### Getting the Code and dependencies
1. Install Anaconda and open a Anaconda Prompt
2. `git clone https://github.com/slaclab/pabv_control`
3. cd pabv_control
4. ./bootstrap.sh 
5. make

### Getting Running
- In Anaconda Prompt  
  - `cd pabv_control/python_client`  
  - `python client_dual.py`
